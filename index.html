Prerequisites

1. Download an Editor

Choose any one of the editor

Atom Editor or Bracket Editor

After installation of one of the editor install its packages as shown below.

2. Install Packages

Atom Packages

To install Atom packages open atom editor then you can goto Packages -> Settings View - > Install Packages/Theme or you can press Ctrl + , then Click on install and then search the following packages in the search bar on the right and press install.

  1. Emmet
  2. atom beautify
  3. color picker
  4. atom-bootstrap3

Brackets Packages

To install Bracketspackages open Bracket editor then you can goto File -> Extensions- > Click on Available then search the following packages in the search bar on the right and press install.

  1. Emmet
  2. Beautify
  3. colorHints
  4. Bootstrap 3 Skeleton
